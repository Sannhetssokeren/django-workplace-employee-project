Проект: Управление сотрудниками и рабочими местами

01. Описание
Этот проект представляет собой веб-приложение на Django, предназначенное для управления информацией о сотрудниках и их рабочих местах.
- Реализованы модели данных для сотрудников и рабочих мест.
- Используются приложения Django для разделения функциональности.
- В проекте применяются принципы Django Coding Style, а также форматтер Black и сортировщик импортов Isort.

02. Структура файлов
- `manage.py`: Скрипт для управления проектом (запуск сервера, миграции и т.д.).
- `myproject/`: Директория основного проекта Django.
  - `settings.py`: Конфигурационный файл проекта.
  - `urls.py`: Основной файл маршрутов.
  - `wsgi.py`, `asgi.py`: Файлы для запуска приложения на сервере.
- `workplaces/`: Приложение для хранения информации о рабочих местах.
  - `models.py`: Модель `Workplace`.
  - `views.py`: Представления для рабочих мест.
  - `apps.py`: Конфигурация приложения `workplaces`.
  - `migrations/`: (Генерируются Django) Файлы миграций БД для `workplaces`.
  - `__init__.py`: Файл инициализации пакета.
- `employees/`: Приложение для хранения информации о сотрудниках.
  - `models.py`: Модель `Employee`.
  - `views.py`: Представления для сотрудников.
  - `apps.py`: Конфигурация приложения `employees`.
  - `migrations/`: (Генерируются Django) Файлы миграций БД для `employees`.
  - `__init__.py`: Файл инициализации пакета.
- `requirements.txt`: Файл с зависимостями проекта (Django, black, isort).
- `README.md`: Описание проекта.

03. Зависимости
- Python 3.x
- Django
- black (для форматирования кода)
- isort (для сортировки импортов)

04. Установка и запуск
Для запуска приложения необходим Python 3.x и установленные зависимости из `requirements.txt`.

05. Установка зависимостей:
1.  **Откройте терминал** в папке проекта (где лежит `manage.py`).
2.  **(Рекомендуется) Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    # Активация (Windows):
    # venv\Scripts\activate
    # Активация (macOS/Linux):
    # source venv/bin/activate
    ```
3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

06. Инициализация базы данных:
1.  **Выполните миграции:**
    ```bash
    python manage.py migrate
    ```

07. Запуск приложения:
1.  **Запустите сервер разработки Django:**
    ```bash
    python manage.py runserver
    ```
    Сервер запустится по адресу `http://127.0.0.1:8000`.

### Просмотр в браузере:
1.  Откройте веб-браузер.
2.  Перейдите по адресу `http://127.0.0.1:8000`.
